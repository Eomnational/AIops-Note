# 1. 计算 Throughput Deprioritization Threshold

## Step 1：计算 Median（中位数）

[  
\text{Median} = \frac{58 + 57}{2} = 57.5  
]

MAD 为中间两个值平均：

## Step 3：计算 Threshold

题目说：

> “providers falling more than 1.5 standard deviations below the median are deprioritized”

在 robust statistics 中，通常会把：

[  
\text{Robust Std} \approx 1.4826 \times MAD  
]

因此：

[  
\text{Robust Std} = 1.4826 \times 18.5 \approx 27.43  
]

最终阈值：

[  
\text{Threshold} = 57.5 - 1.5 \times 27.43  
]

[  
= 57.5 - 41.15  
]

[  
\approx 16.35  
]

# 3. Problem-solving Process & Conclusion

## Process

1. 对所有 throughput 数据排序并计算 median。
    
2. 使用 median absolute deviation（MAD）衡量离散程度，而不是普通标准差。
    
3. 用 robust 标准差近似：
    

[  
1.4826 \times MAD  
]

4. 根据规则：
    

[  
\text{Median} - 1.5 \times \text{Robust Std}  
]

得到 deprioritization threshold。  
5. 将所有 provider throughput 与 threshold 比较。

---

# Final Conclusion

- **Median throughput:** 57.5
    
- **MAD:** 18.5
    
- **Robust standard deviation:** ≈ 27.43
    
- **Deprioritization threshold:** ≈ 16.35 TPS
    
- **Deprioritized providers:** None
    

因为所有 provider 的 throughput 都高于阈值。
